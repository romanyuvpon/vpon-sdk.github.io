# 總覽
---
Vpon Mobile Web SDK 提供多種廣告形式，讓行動網站發佈商可以方便地展示 Vpon 廣告，並賺取收益。 <br>

> **Note**:
>此類型廣告僅支援<strong> `行動裝置`</strong>，使用者透過 PC 瀏覽該網站則廣告不會顯示！
<br>

# 需求條件
---

1. HTML 基本觀念

2. 註冊、申請，並取得 Vpon Mobile Web 版位 ID。<br>
  [我要申請版位 ID](http://console.vpon.com)

3. 修改行動網站程式碼的權限
