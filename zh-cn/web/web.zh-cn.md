# 总览
---
Vpon Mobile Web SDK 可以让各种规模的行动网站发布商方便的显示 Vpon 广告，并赚取收益。


> **Note**:
>此广告仅支援<strong> `行动装置 `</strong>，使用者透过 PC 浏览该网站则广告不会显示！
